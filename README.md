# heart-disease-ml
# ❤️ Heart Disease Prediction ML Project

## 📌 Overview
This project uses Machine Learning to predict whether a person has heart disease based on medical parameters. A Streamlit web app is built for user interaction.

---

## 🚀 Features
- Data preprocessing and cleaning
- Machine Learning model training
- Prediction using Logistic Regression
- Interactive web app using Streamlit

---

## 📂 Files Included
- heartdisease.ipynb → Model training notebook
- heart.csv → Dataset
- app.py → Streamlit web app

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Joblib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
